# Prompt Examples

## Pipeline Skill

```text
Use $syft-news-pipeline to build my profile from syft following and then produce today's personalized briefing.
```

```text
Use $syft-news-pipeline to turn today's Syft-only briefing into a storyline tree and highlight any thin but relevant hobby branches.
```

## Atomic Skills

```text
Use $syft-profile-summary to read syft following and generate following_topics.md, raw_interest_block.txt, and profile_summary.md.
```

```text
Use $syft-daily-briefing to generate today's personalized daily briefing from the existing profile artifacts.
```

```text
Use $syft-storyline-tree to render today's briefing as a trunk-branch storyline tree in markdown, html, and json.
```

```text
Use $syft-storyline-backfill to extend the Fate branch over the last 30 days without rebuilding the whole tree.
```

```text
Use $syft-guidance-rulebook to capture my long-term preference that EVA, Fate, and onsen lines should never be collapsed into one soft-interest bucket.
```

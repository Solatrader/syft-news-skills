# syft-news-skills v0.1.0

## Highlights

- Add one orchestration skill: `syft-news-pipeline`
- Add five atomic skills for profile summary, daily briefing, storyline tree, backfill, and durable guidance
- Add mirrored skill layouts for both GitHub Copilot and Codex installation flows
- Add examples, installation docs, release docs, and repository governance files
- Add build, validation, and release-bundle scripts for reproducible packaging

## Included Skills

- `syft-news-pipeline`
- `syft-profile-summary`
- `syft-daily-briefing`
- `syft-storyline-tree`
- `syft-storyline-backfill`
- `syft-guidance-rulebook`

## Installation Notes

- GitHub Copilot users can copy skills from `.github/skills/`
- Codex users can install from `codex-skills/` or from the release zip

## Validation

Verified locally with:

```powershell
python .\scripts\build_skill_bundle.py
powershell -ExecutionPolicy Bypass -File .\scripts\validate_skills.ps1
powershell -ExecutionPolicy Bypass -File .\scripts\create_release_bundle.ps1 -Version v0.1.0
```

## Known Constraints

- These skills assume `syft` is installed and available
- The workflows target Syft CLI only and do not assume local Python MVP pipeline access
- Default final output language is Simplified Chinese unless the user asks otherwise

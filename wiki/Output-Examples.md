## Storyline Tree Example

This repository includes a full HTML example output:

- [storyline_tree_2026-05-20.html](https://github.com/Solatrader/syft-news-skills/blob/main/examples/artifacts/storyline_tree_2026-05-20.html)

Static preview:

![Storyline tree example](https://raw.githubusercontent.com/Solatrader/syft-news-skills/main/assets/storyline-tree-example.png)

That file demonstrates the kind of deliverable the storyline workflow is trying to produce:

- a readable editorial page
- multiple trunks
- multiple branches per trunk
- merged timelines
- visual distinction between known-pool, topic-top, and search-derived evidence

## Common Artifact Families

### Profile

- `profiles/following_topics.md`
- `profiles/raw_interest_block.txt`
- `profiles/raw_aversion_block.txt`
- `profiles/profile_summary.md`

### Daily Briefing

- `briefings/daily_briefing_<date>.md`
- `briefings/coverage_watchlist_<date>.md`

### Storyline Tree

- `storylines/storyline_tree_<date>.md`
- `storylines/storyline_tree_<date>.html`
- `storylines/storyline_tree_<date>.json`

### Backfill

- `storylines/storyline_backfill_<date>.md`
- optionally updated tree files

### Guidance

- `profiles/user_guidance_rulebook.md`

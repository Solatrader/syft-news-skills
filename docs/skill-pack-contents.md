# Skill Pack Contents

This repository ships six skills in two mirrored layouts.

## Canonical Source

`source-skills/`

Maintain these directories directly:

- `syft-news-pipeline`
- `syft-profile-summary`
- `syft-daily-briefing`
- `syft-storyline-tree`
- `syft-storyline-backfill`
- `syft-guidance-rulebook`

## GitHub Copilot Mirror

`.github/skills/`

Use this mirror when installing project-local skills in a GitHub repository.

## Codex Mirror

`codex-skills/`

Use this mirror when installing skills into Codex from a GitHub repository path or a release zip.

## Included Workflow Coverage

- profile creation from `syft following`
- confirmed long-term editorial guidance
- profile-aware daily briefings
- trunk-branch storyline trees
- branch and trunk backfill without forced reclassification
